# The Missing Minute

YouTube's GenAI roadmap improves every step of the creation journey. Better models, better editing, better post-production. But between "I want to make something" and "here's my prompt," there's no surface at all. That's where intent dies before it becomes a video.

**Live:** [sid-yt.vercel.app](https://sid-yt.vercel.app)

## The thesis

YouTube's creation tools operate on a content axis: improve what gets suggested or generated. Veo 3, Inspiration Tab updates, Edit with AI. Every investment makes the suggestion or the output better.

Nobody is working on the interaction axis: how a first-time creator gets from a vague impulse to a committed prompt. The Shorts text-to-video Create flow offers two paths, both cold starts. Showcase prompt cards (generic, same for every user) or a blank text field ("Describe your idea"). Neither helps you figure out what you want to make.

Disambiguation changes the format. Instead of suggesting a finished prompt, walk the user through 3-4 tappable questions that narrow their intent. 30 seconds, four taps, and the output prompt is unique to them because every word came from their choices.
